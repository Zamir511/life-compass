# 🧭 Life Compass

**Система стратегического планирования жизни**

Life Compass — это не обычный To-Do List. Это инструмент для построения жизни сверху вниз: от жизненных сфер к ежедневным задачам.

## ✨ Возможности

### 📊 Иерархия целей
```
Жизненные сферы
    ↓
Годовые цели
    ↓
Месячные цели
    ↓
Недельные цели
    ↓
Ежедневные задачи
```

### 🎯 Четыре сферы жизни
- **📚 Знания** — обучение, курсы, чтение, навыки
- **🕊️ Духовность** — молитва, размышления, благодарность
- **💰 Финансы** — доход, накопления, инвестиции, карьера
- **💪 Здоровье** — спорт, питание, сон, активность

### 📅 Полноценный календарь
- День / Неделя / Месяц / Список
- Почасовое расписание
- Цветовая маркировка по сферам

### 🌙 Вечернее планирование
Пошаговый мастер для планирования завтрашнего дня:
1. Итоги сегодняшнего дня
2. Активные цели недели
3. Задачи на завтра
4. Расписание по времени
5. Подтверждение плана

### 📈 Статистика и аналитика
- Прогресс по каждой сфере жизни
- Heatmap как у GitHub
- Серии (streaks) по привычкам
- Продуктивность по дням и неделям

### ↻ Трекер привычек
- Ежедневные привычки
- Визуальный календарь выполнения
- Статистика серий

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `⌘/Ctrl + K` | Командная палитра |
| `1` | Главная |
| `2` | Календарь |
| `3` | Цели |
| `4` | Привычки |
| `5` | Статистика |
| `T` | Переключить тему |
| `P` | Спланировать завтра |

## 🛠 Технологии

- **React 19** + **TypeScript**
- **Tailwind CSS 4** — стилизация
- **Zustand** — управление состоянием
- **Framer Motion** — анимации
- **date-fns** — работа с датами
- **LocalStorage** — хранение данных

## 📱 Особенности

- ✅ Полностью оффлайн — работает без интернета
- ✅ Адаптивный дизайн — работает на мобильных
- ✅ Тёмная и светлая тема
- ✅ Экспорт/импорт данных в JSON
- ✅ Данные хранятся локально в браузере
